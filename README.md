## 🎯 프로젝트 개요

여름방학을 맞아 여행지를 찾고 있는 사용자를 위한 맞춤형 여행지 추천 애플리케이션입니다. 
사용자의 예산, 여행 스타일, 관심 분야를 바탕으로 최적의 여행지를 제안하고, 추천된 여행지에서 꼭 가봐야 할 명소들을 소개합니다.

## ✨ 주요 기능

### 1. 예산 기반 추천
- 사용자의 예산에 맞는 여행지 추천
- 예산별 평균 비용 및 가성비 정보 제공

### 2. 여행 스타일 선택
- 액티브한 여행 (트레킹, 스포츠 활동)
- 휴식 중심 여행 (비치, 스파)
- 미식 여행 (로컬 음식, 레스토랑)

### 3. 가볼만한 곳 추천
- 추천된 여행지별 필수 명소 소개

## 🛠️ 기술 스택

- **프론트엔드**: HTML5, CSS3, JavaScript
- **백엔드**: Python / Node.js (선택 시)
- **데이터베이스**: JSON / SQLite (선택 시)

## 📦 설치 및 실행

### 필수 사항
- 웹 브라우저 (Chrome, Firefox, Safari 등)
- Python 3.8+ (백엔드 실행 시)
